조건문

주어진 조건을 만족하거나 만족하지 않을 때 실행할 코드를 달리하기 위해 사용하는 문법

if, else

if문 안에 있는 내용의 참(true) 또는 거짓(false)을 판별하여 참이면 if문 블럭을 실행하고 거짓이면 else문 블럭의 코드를 실행한다. if와 else 사이에는 또 다른 조건 블럭을 추가할 수 있는 else if문을 0개 이상 사용할 수 있다. else와 else if는 생략할 수 있다.

// 변수 num의 값이 3보다 크면 코드를 실행한다.
if (num > 3) {
	console.log('num은 3보다 큽니다');
}

참스러운 값(truthy value)과 거짓스러운 값(falsy value)

조건문에 사용되는 모든 조건이 반드시 true와 false로 구별되지는 않는다. 일부값은 false가 아님에도 조건문에서 마치 false처럼 취급된다.

false처럼 취급되는 값

  • 숫자 0
  • 빈 문자열
  • undefined
  • null
  • NaN

그 밖의 값은 모두 true처럼 취급된다.

switch

비교할 표현식이 특정한 값과 같은지 여러 차례 비교해야 할 때 주로 사용.

/* 좋은 코드는 아니지만, 예제로 봐주세요. */
function 요일(y, m, d) {
	var date = new Date(y, m-1, d), name = '';
	switch(date.getDay()) {
	case 0:
		name = '일';
		break;
	case 1:
		name = '월';
		break;
	case 2:
		name = '화';
		break;
	case 3:
		name = '수';
		break;
	case 4:
		name = '목';
		break;
	case 5:
		name = '금';
		break;
	case 6:
		name = '토';
		break;
	}
	return name;
}
